Installation

Follow the steps below to get started with your eBookingham Mobile App source code:

  1. Create the file with name .env in root directory with content like file env.example
    2. APP_NAME=name_your_app //It will display in some position in your app 
    BASE_DOMAIN=your_domain
    IS_DEBUG=false // if true, you will see log and api when run in debug mode. Default false
    PUSH_NOTIFICATION_KEY='92e18f1c-70a5-4751-98e6-95e80ff4806e'// You don't need to change. This Key will use in next app version
  2. After create .env and fill your information. Need install dependencies and library by run this command
    3. npm -f install
  3. After installed dependencies and library successfully. Let start Metro, run 
    4. npm start -- --reset-cache
  4. Let Metro Bundler run in its own terminal. Open a new terminal inside your React Native project folder. Run the following:
    • For Android
      • npx react-native run-android
      NOTE: Make sure you installed simulator or connected real device
    • For IOS, in root directory of project
      • cd ios && pod install && cd .. && npx react-native run-ios
      NOTE: Make sure you installed simulator or connected real device

Issues

When start running or building your app, you may be get some issues

1. Error 'Multiple commands produce...'

Solution:

  1. In top left of Xcode, select Show the Project navigator and select file name of project (e.g: eBookingham)
  2. In middle bar of Xcode, select Build Phase > Copy Bundle Resources
  3. Delete 16 fonts generated by react-native-vector-icons (photo under). Make sure delete only 16 fonts

2. Warning 'Require cycle: node_modules/rn-fetch-blob/index.js...'

This warning appears automatically when run project in dev mode, it is warning from library rn-fetch-blob and it will always appear after you run command 'npm install', to fix it, follow this:

Solution:

  1. Go to {project_name}/node_modules/rn-fetch-blob/polyfill
  2. In top of 3 file: Blob.js, Fetch.js, XMLHttpRequest.js replace line:
    3. import RNFetchBlob from '../index.js'
    to 
    import {NativeModules} from 'react-native';
    const RNFetchBlob = NativeModules.RNFetchBlob

3. Duplicate resource when build apk after run bundle command

This error auto generate when you run command react-native-bundle....

Solution:

  1. Open file in this directory: {project_name}/node_modules/react-native/react.gradle
  2. In line 143 (after doFirst{}), add this command bellow doFirst{}
    3. doLast {
    def moveFunc = { resSuffix ->
    File originalDir = file("$buildDir/generated/res/react/release/${resSuffix}");
    if (originalDir.exists()) {
    File destDir = file("$buildDir/../src/main/res/${resSuffix}");
    ant.move(file: originalDir, tofile: destDir);
    }
    }
    moveFunc.curry("drawable-ldpi").call()
    moveFunc.curry("drawable-mdpi").call()
    moveFunc.curry("drawable-hdpi").call()
    moveFunc.curry("drawable-xhdpi").call()
    moveFunc.curry("drawable-xxhdpi").call()
    moveFunc.curry("drawable-xxxhdpi").call()
    moveFunc.curry("raw").call()
}

FAQs

1. How to add new language in mobile app?

  1. Find file in the directory src/config/languages/index.js
  2. To add new language, duplicate any section in file and translate content in inverted comma
    3. Ex: Add new language like: Arabic. Follow this step:
    1. Go to admin dashboard of your website. Open Tools > Languages
    2. See column Locale in right panel and make sure new language added in mobile app have a key in this column. In this case, i will choose key is ar
    3. Return the mobile app project, after choose key language. I will collapse content and duplicate new section from en
    4. After duplicate, change key of this language from en to ar
    5. Start translate content of each key in new section ar
      6. NOTE: In each section, we have key is: isRtl. If your language support and require Right to left layout. Please change value of this key from false to true

2. How to change icon inside splash screen?

  1. In directory src/assets/image, replace 3 files app.png, icon_app.png, logo.png with your icon (make sure you use same name and file extension with old file)
  2. Open project in Terminal, run this command this root directory of project: npx react-native generate-bootsplash src/assets/image/app.png --background-color=FFFFFF
    3. NOTE: You can change color with option --background-color=your_color_lor. See more option here

3. How to change display name when install in real device or simulator?

  1. Android
    1. Open file android/app/src/main/res/values/string.xml and change value inside tag string
      2. Ex: Change eBookingham to Your App Name

4. How to change bundle id?
